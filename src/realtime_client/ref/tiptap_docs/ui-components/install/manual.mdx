## Configure path aliases

Configure the path aliases in your `tsconfig.json` file.

```json
{
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@/*": ["./*"]
    }
  }
}
```

The `@` alias is a preference. You can use other aliases if you want.

## Configure style

Follow the [style setup guide](/ui-components/getting-started/style) to ensure your editor and components render correctly.

### Thats it!

You can now use Tiptap UI Components in your project without any additional setup. Just import the components you need and start building your editor.
